# 🛠️ Xcode Features Sample
A SwiftUI sample project showcasing useful Xcode features — previews, bookmarks, source control, and more.

---

## 🤔 What this is
This project is a companion to a tour of Xcode's built-in features. It demonstrates how to use Xcode Previews effectively, work with the Source Control navigator, set bookmarks, use the Find navigator, and navigate large codebases efficiently. A practical reference for getting more out of the IDE you use every day.

## ✅ Why you'd use it
- **Xcode Previews** — Shows how to set up multiple preview configurations and use `#Preview` macros
- **Source control** — Demonstrates committing, branching, and viewing diffs directly in Xcode
- **Productivity features** — Covers bookmarks, code folding, the minimap, and quick navigation shortcuts

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/XcodeFeatures_Sample.git
cd XcodeFeatures_Sample
```

### 2. Open in Xcode
Double-click `xcodefeatures.xcodeproj`.

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team** — select your team.

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique reverse-domain ID.

## 🛠️ Notes
- No third-party dependencies required.
- If you see a code signing error, verify Team and Bundle ID are set.

## 📦 Requirements
- Xcode 15+
- iOS 16+
